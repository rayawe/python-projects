## Python Projects

This repository contains simple Python projects created for practice and system monitoring.

### System Disk & Memory Checker
- Python script to check disk usage and memory availability on Windows systems.
- Helps identify basic performance and storage issues during troubleshooting.
- Built using Python standard libraries and psutil.

### Simple To-Do List Program
- Console-based Python program to add, view, and remove tasks.
- Demonstrates basic Python logic, lists, and user input handling.

### Simple User Management System
- Python program to simulate basic user management operations.
- Supports adding, viewing, and deleting users from a local file.
- Demonstrates file handling and basic CRUD operations.
